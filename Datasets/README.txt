The input dataset should be a text file.
Each line is a record, with four integer separated by space or tab: user_id item_id rating timestamp
We offer the Epinions dataset after the cleaned up preprocess in Dataset\Epinions\epinions.txt
The Movielens100k dataset can be downloaded at https://grouplens.org/datasets/movielens, copy the u.data file in it to Dataset\MovieLens100k\ and our program will be able to load it.
